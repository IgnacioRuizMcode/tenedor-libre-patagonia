> **Sistema completo de gestión de reservas para buffet premium en Puerto Montt, Patagonia**

## ✨ Características

### 👤 Usuarios No Autenticados
- ✅ Reservar mesa sin necesidad de registro
- ✅ Confirmación por WhatsApp
- ✅ Validación de fechas y horarios

### 👥 Usuarios Autenticados
- ✅ Login con Google o Email/Password
- ✅ Ver historial de reservas
- ✅ Modificar cantidad de personas
- ✅ Cancelar reservas
- ✅ Agregar evento a Google Calendar

### 👑 Administrador
- ✅ Dashboard con estadísticas en tiempo real
- ✅ Filtrar reservas por fecha, cliente o estado
- ✅ Modificar cantidad de personas
- ✅ Cambiar estado de reservas
- ✅ Exportar reservas a CSV
- ✅ Notificaciones por email

## 🛠️ Tecnologías

| Categoría | Tecnologías |
|-----------|-------------|
| **Frontend** | React 18, React Router, Axios, Tailwind CSS |
| **Backend** | Node.js, Express, MongoDB, Mongoose |
| **Autenticación** | Passport.js, Google OAuth, JWT |
| **Email** | Nodemailer |
| **UI/UX** | Lucide Icons, React DatePicker |
